# EventApp — Android Event Management App

A native Android app for managing events with full CRUD functionality. Built independently as a university project.

## What it does
- Browse and filter a list of events
- Add new events with details (title, date, location, description)
- Edit and delete existing events
- Persistent local storage via Room database

## Technologies
- **Kotlin** — Android development
- **Android SDK** — UI and navigation
- **Room Database** — local data persistence
- **RecyclerView** — efficient list display

## Background
Developed independently from scratch as a university project. Full development cycle: architecture design, UI implementation, database layer, and testing.
